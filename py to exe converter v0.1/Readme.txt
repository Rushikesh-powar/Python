Py2Exe Converter

1.This application converts python scripts (.py) to executable's (.exe). 
2.This application also provids the user an GUI which prevents users work    to put all the commands by himself.

3.This application uses an module named "pyinstaller" for the          converstion.

4.This module provides user provision to combine all the file and required   libraries into a single executable file.
  By default all the file and libraries are unpacked with .exe file.

5.If you application is a GUI or has any type of interface than it also      provides -w(window) option.
